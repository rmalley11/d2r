Name	Manchester EncoderSynopsis Description	This program converts plain binary into a manchester encoded data stream. As of now, the program can only convert in one direction, but converting from manchester to plain binary should be implemented soon. This program is an exercise made to mimic a sensor outputting information and the subsequent encoding done to that information in order to make a proper waveform. Inputs should be in the form of a binary string passed as an argument to the command. To Do
